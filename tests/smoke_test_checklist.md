# Smoke Test Checklist

Use this quick checklist before demo/submission:

- [ ] `docker compose config` is valid
- [ ] `docker compose ps` shows expected services running
- [ ] `run.bat` exits `0`
- [ ] `dbt test` passes
- [ ] markdown report generated
- [ ] PDF report generated
- [ ] figures generated and current timestamp updated
- [ ] `docs/architecture.md` and `docs/decisions.md` present
- [ ] `docs/er_diagram.png` present
